# legacy/ — quarantined dead code

This directory holds code that is **not imported or executed by the live system**, moved here on 2026-06-10 per `docs/REFACTORING_PLAN.md` Step 1 ("Карантин баласту"). It is kept only for reference/history: `communicator.py` (XBee DigiMesh link, superseded by the Telnet command interface), `ultrasonic_sensor.py` (Jetson GPIO ultrasonic altimeter, replaced by the LIDAR sensor), `targetTracker.py` (unused tracking helper), `modules/` (aspirational abstract interfaces that were never implemented), and `convertor/` (one-off conversion/model scripts). Verified by grep at move time: nothing under `src/`, `sky_anchor/`, `gazebo/`, `examples/`, or the entry point references any of these modules. Do not import from here; if something is needed again, move it back deliberately.
